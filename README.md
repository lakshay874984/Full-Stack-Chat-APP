🚀 Chat Application

🔹 Overview
A full-stack real-time chat application that allows users to send messages instantly, upload images, and see online user status.
---
## 🛠 Tech Stack
  - React.js
  - Node.js
  - Express.js
  - MongoDB
  - Socket.io
  - JWT Authentication
  - Cloudinary
